🔧 Setup Instructions

  1. Go to Google Cloud Console
  2. Create a project, enable Gmail API
  3. Go to Credentials → Create OAuth 2.0 Client ID → Desktop app
  4. Add redirect URI: http://localhost:3000/api/auth-callback
  5. Download credentials.json and put it in server/ folder
  6. Run npm start inside server/ folder
